# Glossary

- **STIG**: Security Technical Implementation Guide.
- **XCCDF**: Extensible Configuration Checklist Description Format.
- **OVAL**: Open Vulnerability and Assessment Language.
- **CAT I/II/III**: Severity categories (high/medium/low).
- **OpenSCAP**: Open source SCAP compliance toolkit.
- **CAC**: ComplianceAsCode content repository.
- **Kickstart/Preseed**: Automated installation configurations.
