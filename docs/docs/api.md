# API Documentation

StreamGuard exposes an interactive Swagger UI at:

```
http://<server-ip>:8000/docs
```

If you are running locally, replace `<server-ip>` with `localhost` or your LAN IP.
This page includes all API endpoints, request/response schemas, and example payloads.

## Quick Link

- Swagger UI: `http://<server-ip>:8000/docs`
