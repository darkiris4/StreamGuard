## Security Policy

StreamGuard takes security seriously. If you discover a vulnerability,
please report it privately.

## Reporting a Vulnerability

Preferred:
- Use GitHub Security Advisories for private reporting.

If Security Advisories are not available:
- Contact the maintainer via https://github.com/darkiris4

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any proof-of-concept code or screenshots

Do not open public issues for security vulnerabilities.

## Supported Versions

Security updates are provided for the latest release and the `main` branch.
Older versions may not receive fixes.

## Response Timeline

We aim to:
- Acknowledge reports within 72 hours
- Provide a remediation plan or fix timeline within 7 days
